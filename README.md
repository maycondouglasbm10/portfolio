# Portfólio

Este projeto é um Portfólio Web, desenvolvido com **HTML5, CSS3 e JavaScript**, com foco em boas práticas de desenvolvimento front-end, como estrutura semântica, responsividade e design limpo.
O objetivo é apresentar minhas habilidades, projetos e formas de contato de forma moderna e acessível, servindo como vitrine profissional.

## 🧠 Funcionalidades

- 📌 Apresentação pessoal (sobre mim)
- 💼 Listagem de projetos com descrição
- 📱 Layout totalmente responsivo (mobile e desktop)
- 🎯 Navegação simples e intuitiva
- ✉️ Formulário de contato com validação básica
- ⚡ Interações com JavaScript (menu, animações, etc.)
- ♿ Estrutura semântica e acessível

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Finalidade |
|-------------|-------------|
| 🧩 **HTML5** | Estrutura do formulário e semântica dos campos |
| 🎨 **CSS3**  | Estilização, espaçamento e responsividade |
| **JAVASCRIPT**| Interatividade e manipulação do DOM |
